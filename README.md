# PSTokenlab
## Dependencias servidor API:
````
$ npm install express
$ npm install mysql
$ npm install cors
$ npm install body-parser
$ npm install nodemon
````

## Dependencias Front:
````
$ npm install @angular/cli -g
````



## Rodar servidor angular:

````
$ ng serve
````
## Rodar servidor API:

````
$ nodemon run
````
